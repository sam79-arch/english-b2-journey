## ⏰ Daily Schedule (Mon–Fri, 90 mins Flexible)

| Duration | Module | Activity | Time Allocation |
|---|---|---|---|
| 10 min | 🐛 Debug Review | Re-read yesterday's Bug Log — avoid repeating errors | ~10% |
| 30 min | 📥 Input | Study 1 GIU unit — understand the logic/syntax | ~30% |
| 25 min | ⚙️ Processing | VIU vocabulary + exercises | ~25% |
| 15 min | 📤 Output | Write 5–10 sentences in IT context | ~20% |
| 10 min | 🗣️ Speaking + 📦 Commit | Say sentences aloud (ChatGPT/Gemini) → `git commit` | ~15% |

## ⏰ Saturday Schedule (2 hours — Deep Review Sprint)

| Time | Activity |
|---|---|
| 08:00–08:45 | Review week's grammar — finish remaining GIU exercises |
| 08:45–09:30 | Review vocabulary + Word Formation + update Flashcards |
| 09:30–09:50 | Integrated skill practice (Writing or Reading depending on phase) |
| 09:50–10:00 | Write **Weekly Review file** → `git commit` |

## ⏰ Sunday Schedule (2 hours — Production Test)

| Time | Activity |
|---|---|
| 08:00–09:15 | Grammar test or mini IELTS mock |
| 09:15–09:45 | Mark test, **update Bug Log**, analyse root cause of each error |
| 09:45–10:00 | Paste writing into **NotebookLM** → Preview next week → `git commit` |

---

## 🚨 Fallback Rule

> **Miss 1 weekday:** Carry half the content to Saturday, half to the next evening.
> **Miss Saturday or Sunday:** Commit at least 1 file (min. 5 sentences) — keep the GitHub streak green.
> **Never:** Skip 2 consecutive days. 2 days in a row = momentum reset.
