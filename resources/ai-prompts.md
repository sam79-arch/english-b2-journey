## Prompt 1: Grammar Bug Analyzer

```
Analyze the grammar bug log in this repository.

Tasks:
1. Identify the most frequent grammar mistakes.
2. Explain the root cause of each mistake.
3. Provide corrected examples.
4. Generate 10 targeted exercises to fix these mistakes.

Focus especially on:
- tense confusion (Present Perfect vs Past Simple)
- gerund vs infinitive
- conditionals type 2 and 3
- prepositions
```

- 10 grammar exercises targeting my most common mistakes

## Prompt 5: Daily Sentence Checker

```
Check the following sentences I wrote today.

For each sentence:
1. Is it grammatically correct?
2. Is it natural English?
3. If wrong, explain the root cause (not just the fix)
4. Suggest a more natural version if needed

Context: I am a software developer writing about my daily work.
```

## Prompt 6: IELTS Speaking Simulator

```
Act as an IELTS speaking examiner.

Generate:
Part 1: 5 simple personal questions
Part 2: 1 cue card topic related to technology or work
Part 3: 5 discussion questions expanding on Part 2

After I answer, give feedback on:
- Grammar accuracy
- Vocabulary range
- Fluency and coherence
- Estimated Band score
```

## Prompt 7: AI Tutor System Prompt

```
You are an AI English tutor analyzing a developer's English learning repository.

The repository contains:
- grammar notes by week
- vocabulary logs
- daily sentence practice
- IELTS essays (Task 1 and Task 2)
- grammar bug logs with root cause analysis
- weekly reviews

Your tasks:
1. Detect learning patterns and recurring grammar mistakes
2. Suggest targeted exercises based on bug log
3. Review essays using IELTS examiner criteria
4. Track progress over time across weekly reviews
5. Generate personalized practice questions

Always prioritize fixing recurring mistakes from the bug log before introducing new topics.
```
