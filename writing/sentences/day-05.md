# Day 05 - Writing Practice (2026-03-09) — Present Perfect Continuous vs Simple

## Context: IT & Development
Focus: Unit 9–10 — Process (how long) vs Result (completion)

---

## 10 Sentences: Present Perfect Continuous vs Simple

### Present Perfect Continuous (I have been doing) — Progress, Duration, "How long"

1. **Original:** I am working on this bug for 2 hours.
   **Corrected:** I **have been working** on this bug for 2 hours.
   *Note:* "For + time period" indicates ongoing process → Present Perfect Continuous.

2. **Original:** She works on the new feature since Monday.
   **Corrected:** She **has been working** on the new feature since Monday.
   *Note:* "Since + point in time" marks the start of ongoing action → Present Perfect Continuous.

3. **Original:** We test the database and we find 3 critical errors.
   **Corrected:** We **have been testing** the database and **have found** 3 critical errors.
   *Note:* "Have been testing" = ongoing process; "have found" = results so far.

4. **Original:** The server is crashing for the last week.
   **Corrected:** The server **has been crashing** for the last week.
   *Note:* Duration in past extending to now → Present Perfect Continuous.

5. **Original:** My colleague develops this API and he finishes it yesterday.
   **Corrected:** My colleague **has been developing** this API and **has just finished** it.
   *Note:* Mix of process (developing) and recent result (finished).

---

### Present Perfect Simple (I have done) — Result, Experience, Completion

6. **Original:** I finishing the project presentation.
   **Corrected:** I **have finished** the project presentation.
   *Note:* Recent action with present result → Simple. The work is done now.

7. **Original:** Have you ever attended an online conference?
   **Corrected:** **Have you ever attended** an online conference?
   *Note:* Experience/lifetime question → Present Perfect Simple. ✓ (Already correct)

8. **Original:** They already upload the code to GitHub.
   **Corrected:** They **have already uploaded** the code to GitHub.
   *Note:* "Already" + recent completion → Present Perfect Simple.

9. **Original:** The IT team set up three new servers.
   **Corrected:** The IT team **has set up** three new servers.
   *Note:* Completed action; result visible now → Present Perfect Simple.

10. **Original:** I have been fixing this error and I fix it finally.
    **Corrected:** I **have been fixing** this error and **have finally fixed** it.
    *Note:* First part = ongoing struggle; second = completion. Use both tenses to show process then result.

---

## Summary: When to Use Each

| Present Perfect Continuous | Present Perfect Simple |
|---|---|
| Focus on **how long** / **process** | Focus on **result** / **completion** |
| "How long have you been...?" | "Have you ever...?" |
| "I've been working for 3 hours" | "I've finished the report" |
| Still happening (usually) | Finished (usually) |
